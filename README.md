## About

Backroads is an indie game project focused on exploring small-town life, vehicle customization, and open-world gameplay. Players will be able to explore the backroads, earn money, repair and modify vehicles, and build their own story in a fictional Newfoundland-inspired community.

## Features

- Vehicle customization and upgrades
- Property and progression systems
- Dynamic weather
- Object interaction and item systems
- Exploration of a rural open world

## Built With

- Unity
- C#
- Blender
- Aseprite

## Development

This project is currently in early development. Features and systems are actively being prototyped and refined.
